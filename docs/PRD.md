# PRD: Pomodoro 타이머 웹 앱

## 1. 프로젝트 개요

집중력 향상을 위한 개인용 Pomodoro 타이머 웹 앱. 작업(25분)/휴식(5분) 사이클을 관리하고 완료 세션을 기록하여 생산성 향상을 돕는다. 초기에는 개인 로컬 사용을 목적으로 하며, 완성도에 따라 공개 배포를 고려한다.

---

## 2. 목표

- 복잡한 설정 없이 즉시 사용 가능한 Pomodoro 타이머 제공
- 완료 세션 기록으로 성취감 및 동기 부여
- 향후 배포를 고려한 확장 가능한 구조

---

## 3. 타겟 사용자

- **주 타겟**: 집중력 향상이 필요한 지식 근로자, 학생, 프리랜서
- **부 타겟**: Pomodoro 기법을 처음 접하는 시간 관리 초보자

---

## 4. 핵심 기능 (MVP)

### 4.1 타이머
- 작업 세션: 기본 25분 카운트다운
- 휴식 세션: 기본 5분 카운트다운 (4세션마다 긴 휴식 15분)
- 시작 / 일시정지 / 리셋 제어
- 세션 전환 시 시각/청각 알림 (브라우저 알림 + 알람음)

### 4.2 세션 기록
- 완료된 Pomodoro 세션 횟수 표시
- LocalStorage를 통한 날짜별 기록 유지 (새로고침 후에도 보존)
- 오늘의 완료 횟수 시각적 표시

### 4.3 설정
- 작업/휴식/긴 휴식 시간 커스터마이징
- 알림음 on/off 토글
- 다크/라이트 테마 전환

---

## 5. 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 타이머 정확성 | 백그라운드 탭에서도 오차 1초 이내 |
| 데이터 지속성 | 브라우저 새로고침 후에도 세션 기록 유지 |
| 성능 | 초기 로딩 1초 이내 |
| 호환성 | 모던 브라우저 (Chrome, Firefox, Safari, Edge 최신 2버전) |
| 반응형 | 모바일/태블릿/데스크톱 모두 지원 |

## 8. UX 원칙

1. **단순성 우선**: 핵심 기능(시작/정지)이 화면 중앙에 위치, 부가 기능은 숨김
2. **신뢰성 보장**: 타이머 정확성과 데이터 지속성 최우선
3. **즉각적 피드백**: 세션 전환마다 명확한 시각/청각 신호
4. **개인화**: 시간 설정 커스터마이징으로 다양한 작업 유형 대응

---

## 9. 주요 UX 리스크 및 대응

| 리스크 | 우선순위 | 대응 방안 |
|--------|---------|----------|
| 백그라운드 탭에서 타이머 오차 | 높음 | Page Visibility API + 절대 시간 기반 타이머 |
| 새로고침 시 데이터 손실 | 높음 | LocalStorage에 실시간 상태 저장 |
| 알림 놓침 | 높음 | 브라우저 알림(Web Notification API) + 알람음 병행 |
| 인터페이스 과잉 복잡도 | 중간 | MVP는 핵심 기능만, 설정은 별도 패널 |

---

## 10. 성공 지표

- 시작한 세션의 완료율 > 80%
- 7일 내 재사용율 측정 (배포 후)
- 하루 평균 완료 세션 수 추적

---

## 11. 향후 확장 고려사항

- PWA 전환 (오프라인 지원, 모바일 설치)
- 작업 이름 태깅 및 프로젝트별 통계
- 주간/월간 리포트
- 팀 기능 (B2B 피벗 시)
